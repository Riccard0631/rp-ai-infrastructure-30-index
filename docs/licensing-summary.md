# RPAI30 Draft Licensing Summary

## Current Status

RPAI30 is currently available as an informational benchmark and data product.

It is not an ETF, fund, advisory service, regulated benchmark, or investment product.

## Free Public Use

The following uses are intended to be permitted without a commercial license:

- viewing the public dashboard
- reading the methodology
- reading component data
- viewing public API responses manually
- linking to the public site
- discussing the index for educational or informational purposes

## Commercial Use Requiring Permission

The following uses should require a written commercial agreement:

- republishing the index level at scale
- using the API in a commercial product
- using the index in paid research
- white-labeling the dashboard
- using RPAI30 branding in marketing
- distributing RPAI30 data through a third-party platform
- creating derived indices or model portfolios using RPAI30 branding

## Regulated Product Use

The following uses are not permitted without separate legal, regulatory, data, index administration, and licensing review:

- ETF
- ETP
- fund
- certificate
- structured product
- derivative
- financial contract
- benchmark for performance fees
- regulated investment product

## Evaluation License Concept

Potential ETF/ETP issuers, index administrators, data vendors, and research partners may review RPAI30 for evaluation purposes.

An evaluation package may include:

- methodology
- component file
- historical backtest
- current history
- REST API schema
- governance draft
- factsheet

## Required Next Steps For Product Use

Before product use, the parties should resolve:

- benchmark administrator status
- jurisdiction and regulatory treatment
- calculation agent responsibilities
- market-data licensing
- corporate action handling
- index name and trademark ownership
- methodology change policy
- restatement policy
- licensing fees
- liability limits
- termination and fallback provisions

## Disclaimer

This document is a commercial planning draft, not legal advice.
